   ```
   使用 iPhone 17 Pro Max 模拟器

   - 播放中状态
   - Sleep Timer 设置
   ```

2. **创建隐私政策**  
   ```
   最简单方法：
   1. 在 GitHub 创建一个 repository
   2. 将提供的隐私政策模板放入 README.md
   3. 启用 GitHub Pages
   4. 使用生成的 URL
   ```

3. **在 App Store Connect 创建应用**  
   ```
   1. 登录 appstoreconnect.apple.com
   2. 点击 "My Apps" > "+" > "New App"
   3. 填写基本信息
   4. 使用 Bundle ID: app.Little-Moon
   ```
